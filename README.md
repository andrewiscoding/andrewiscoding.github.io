# Heavy Dose 2004 Band Site

A static band website starter for GitHub Pages.

## How to use

1. Put your real images inside `/assets`
   - `hero.jpg`
   - any press photos / cover art
2. Edit the links in `index.html`
3. Commit and push to GitHub
4. In GitHub: Settings → Pages → Deploy from branch → `main` → root

## Notes

This is intentionally simple: HTML, CSS, and a tiny bit of JS.
No build step, no npm, no backend.
